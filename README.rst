Install
-------

From Source
***********

Installing from source is as easy as

.. code-block:: bash

    python setup.py install

See required dependencies via the online documentation.

Build the Docs
--------------

Requirements
************

See `doc/environment.yml`

Build and Serve
***************

.. code-block:: bash

    cd doc
    make html

Then point you browser to `http://127.0.0.1:8000/`.
